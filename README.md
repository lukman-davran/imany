# Posebna Poruka - Personalizovana Web Stranica

Romantična i personalizovana web stranica posvećena posebnoj osobi.

## Tehnologije

- HTML5
- CSS3
- Responsive Design

## Struktura Projekta

```
Moj_sajt/
├── index.html          # Glavna HTML stranica
├── images/            # Folder sa slikama
│   └── background.jpg.jpg
└── README.md
```

## Deployment

Sajt je hostovan na Netlify i povezan sa GitHub repository-jem.
